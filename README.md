# BCP AutiFraud code challenge

This repo contains an simple implementation to solve the BCP Antifraud code challenge.

Stack technology:
* .Net 8
* Docker for containers
* Kafka
* InMemoryDb

## Getting Started

You will need a certain tools in order to locally install the app

### Prerequisites

* Docker for Windows
* Visual Studio 2022
